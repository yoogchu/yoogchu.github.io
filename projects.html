    Clean Water Reporter

    A JavaFX application loosely based on a NASA challenge. It supports finding clean drinking water and reporting on testing results. In many parts of the world, access to clean water is a severe problem. Many times, water is available, but people do not know where to get it. Other times, water may appear clean, but has unacceptable levels of contaminants.
    This application supports user creation/authentication/management, admin access, logging, submitting a water report, data visualization of water reports, and database logging.

    I'm working with PostgreSQL and JavaFX to create the controllers for connecting the model and view together in a MVC software architectural pattern. The team is also following Agile software development practices.

    Tweety McTweetBot

    A twitter bot that currently allows users to mention the bots id: '@pythonmcbotty', and will reply to specific commands. A current command involves responding to a flight number with current flight status, gate of the flight, and departure time. Another command calculates the time needed to leave the user's current location by analyzing live traffic data in the Google Map's API as well as the TSA lines in the given airport.

    I created the bot using Python, Node.js, the Google Maps API, and the Delta Airlines API.

    Pacman Artificial Intelligence

    A python program that simulates artificial intelligence to solve certain goals for Pacman, whether it be finding the fastest route to a certain location, collecting the most points from food, or a combination of both.

    I integrated search algorithms such as depth first search, breadth first search, A*, as well as created several heuristics to create an optimal algorithm that finds the most efficient path to the goal state.

    Turbine Engine Cycle Analysis

    A MATLAB program that calculates the performance parameters for a generic jet engine cycle. It has the capability of turning on and off specific components based on the performance requirements.
    Components include: Diffuser, Bypass Fan, Compressor, Fuel Pump, Main Burner, Turbine, Turbine Mixer, Fan Turbine, Afterburner, Core Nozzle, Fan Nozzle, Nozzle Mixer, and Combined Nozzle.

    I helped derive the equations with a team of 2 others and designed the code structure as well as output the performance parameters to a formatted text file.
